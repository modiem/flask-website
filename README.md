# Flask Website
A website where user can login, save and delete notes.

## Features
- Flask (Blueprint, render_template, flash, redirect, url_for, login manager)
- SQLAlchemy (Database Models)
- Jinja templating language
- Bootstrap
- password hash
- JavaScript (fetch)
